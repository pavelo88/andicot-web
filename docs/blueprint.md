# **App Name**: ANDICOT Digital

## Core Features:

- Dynamic Public-Facing Website: Render interactive and visually appealing content including hero sections, service offerings, brand partnerships, and company statistics, all driven by a centralized content store.
- Admin Dashboard & Authentication: Provide a secure login and a dedicated interface for administrators to manage all aspects of the application's content and data.
- Comprehensive Content Management System (CMS): Enable administrators to perform CRUD (Create, Read, Update, Delete) operations on website content such as hero elements, detailed service descriptions, company statistics, and brand logos. This feature will heavily utilize Firebase Firestore for data storage and Firebase Storage for handling all media assets, such as service images and brand logos.
- Integrated Customer Relationship Management (CRM): Manage incoming prospect leads, track their status (new, in progress, closed), and maintain communication history directly within the admin dashboard. Leads will be captured via the public-facing contact form and stored in Firebase Firestore.
- AI-Powered Lead Intelligence: Leverage generative AI to analyze incoming lead inquiries, suggest response drafts, and automatically generate detailed commercial proposals, significantly streamlining the sales process with an AI tool.
- AI Marketing & Content Automation: Utilize generative AI tools to assist administrators in generating SEO meta-descriptions, keywords, and creative social media posts for individual services, ensuring optimal visibility and engagement.
- Interactive AI Chatbot: Implement a publicly accessible AI chatbot capable of answering user queries about ANDICOT's services and technology ecosystems, enhancing user experience and immediate information retrieval with a conversational AI tool.

## Style Guidelines:

- Dark color scheme to evoke sophistication and a technological aesthetic.
- Primary action color: A vibrant lime green (#A4C851) is used for call-to-action elements, icons, and branding highlights, symbolizing innovation and vitality.
- Background color: A deep, almost-black blue-purple (#0a0b1a) forms the primary canvas, providing a serious and high-tech backdrop for content.
- Accent UI color: A slightly brighter dark blue-purple (#242853) is employed for card backgrounds and secondary UI elements, creating depth within the dark palette.
- Headline and bold text font: 'Roboto' (sans-serif) for modern clarity and readability across various weights.
- Body and accent text font: 'Playfair Display' (serif) for an elegant and inviting feel, especially for descriptive paragraphs.
- Use a cohesive set of modern outline icons from Lucide React to visually represent services and UI actions, maintaining a clean and professional look.
- Responsive, grid-based layouts ensure optimal viewing on diverse devices. A 'glass-card' aesthetic provides a sophisticated, translucent effect for UI components. Key elements like the navbar and WhatsApp button remain fixed for easy access.
- Subtle, smooth transitions on hover effects and state changes (`transition-all`). Infinite scroll animations for brand carousels add a dynamic feel. Loading indicators incorporate a 'spin' animation for better user feedback.
